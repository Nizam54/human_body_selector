# human_body_selector

Human body part selection with desired color.

## Getting Started

```dart
HumanBodySelector(
map: Maps.HUMAN,
onChanged: (bodyPart, active) {},
onLevelChanged: (bodyPart) {},
multiSelect: true,
toggle: true,
height: MediaQuery.of(context).size.height,
width: MediaQuery.of(context).size.width-100,
),
```
